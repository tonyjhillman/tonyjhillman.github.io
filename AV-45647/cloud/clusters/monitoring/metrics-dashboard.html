Database Metrics

    +
    On the Monitoring dashboard, you can see metrics for each database presented as charts. You can customize the metrics shown, adjust timescales, and zoom in on chart areas.

    Overview

    Each database has metrics available on the Monitoring dashboard. The metrics available on the Monitoring dashboard provide you with insight into database performance and can help identify runtime outliers. This dashboard can also help investigate intermittent issues by adjusting timescales and viewing multiple metrics on one page.

    Viewing the Monitoring Dashboard

    The Monitoring dashboard is the first thing you see when you open a database. Located under the Monitoring tab in an open database, this dashboard provides you with a customizable collection of real-time metrics for the current database.

    If you don’t see any metrics listed, this page shows the Choose Metrics message. In this message, click the Data Service link to populate the Monitoring dashboard with the default metrics set.

    The Monitoring dashboard includes the following:

    Services

    Near the top of the Monitoring dashboard is the Services drop-down menu, which lists the metrics shown on the dashboard. You can also use this drop-down to find and add new metrics to the dashboard and remove existing ones.

    Timescale

    The timescale filter allows you to quickly switch between preselected time ranges moving backward from the current time.

    Metric tile

    The Monitoring dashboard shows each metric as a tile. A tile includes a title, a graph with labeled X and Y axis, data lines, and a close icon .

    Using Metrics

    Each metric tile on the Monitoring dashboard shows you a chart representing data from the chosen time frame.

    Moving your mouse pointer over a metric chart provides date, time, and resource information for the part of the chart you are pointing at. You can also zoom in on areas of a chart by dragging your pointer over the region you want to focus on.

    Zooming in on a chart updates the timeframe to match your current selection for all metrics on the dashboard.

    Add Metrics

    You can customize the Monitoring dashboard to mix and match the metrics that you want to track.

    To add metrics to the Monitoring dashboard:

    1. Using the Services list near the top of the Monitoring dashboard, select the metric you’re adding to the dashboard.

      Repeat this step to add multiple metrics.

    The Monitoring dashboard adds new metric tiles to the bottom of the page.

    Remove Metrics

    You can remove any or all metrics from the Monitoring dashboard using the following methods:

    • In the metric tile you want to remove, click the close icon in the corner.

    • In the Services list, beside the name of a metric you want to remove, click the small close icon .

    Choose Timeframes

    By default, the Monitoring dashboard shows metric data from the past hour. This updates in real-time. Using the timescale buttons below the Service list, you can choose from the following preselected time ranges:

    • 30 minutes

    • 1 hour

    • 2 hour

    • 1 day

    • 2 days

    • 7 days

    • 30 days
